hexo for markdown static blog

  1. install hexo: npm install -g hexo-cli
  2. update hexo: npm update hexo-cli -g

测试

  • markdown文件存储位置：/opt/hexo-blog/source/_posts
  • 生成静态文件：hexo generate

    lighttpd -f /usr/local/etc/lighttpd/lighttpd.conf
